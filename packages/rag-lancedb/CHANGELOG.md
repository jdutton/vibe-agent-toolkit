### Added

- Initial release of LanceDB RAG implementation
- `LanceDBRAGProvider` implementing both `RAGQueryProvider` and `RAGAdminProvider`
- Resource-level change detection with content hashing
- Readonly and admin modes
- Integration with ResourceRegistry for markdown resources
- Full vector search with metadata filtering
